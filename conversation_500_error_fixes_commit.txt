## Summary
Fixed critical 500 Internal Server Errors occurring when users open the chat widget by implementing defensive programming and proper error handling in widget controllers.

- Fixed NoMethodError in conversation_params when inbox is nil for new visitors
- Enhanced object validation in conversation lookup methods
- Improved error handling in update_last_seen endpoint
- Added graceful degradation for missing conversation objects

## Technical Changes

### BaseController (app/controllers/api/v1/widget/base_controller.rb)
